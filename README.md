# projet-personnel
Ce projet personnel est une application web de gestion de clients et d'activités. Il s'agit d'une plateforme permettant de créer, lister et gérer des clients ainsi que leurs activités associées. Le projet comprend une base de données SQL, des interfaces utilisateur pour les formulaires et des pages de liste, avec un système backend pour traiter les requêtes serveur.

### Points Fonctionnels Principaux
- Gestion des clients (création, consultation, listing)
- Gestion des activités associées aux clients
- Système de formulaires pour l'ajout de nouvelles données
- Interface de visualisation des listes de clients et d'activités
- Système d'abonnement/souscription pour les clients
- Affichage dynamique avec chargement asynchrone (loader)

### Pile Technologique
- **Frontend:** JavaScript (39.5%), CSS (11.6%)
- **Backend:** PHP (29.1%)
- **Base de données:** SQL (fichier bd_tp2.sql)
- **Markup:** Hack (19.8%) - fichiers PHP avec templating
